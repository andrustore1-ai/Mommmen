تحديث V6 - إصلاح عاجل

ما تم إصلاحه:
1) إزالة سبب ظهور كود JavaScript كنص داخل صفحات النظام مثل الكاشير.
   السبب كان وجود وسم script داخل قالب طباعة الفاتورة، وهذا كان يكسر سكربت الصفحة.
2) إصلاح زر القائمة الجانبية في المنيو الرقمي بإضافة طبقة حماية تفتح/تغلق القائمة حتى لو تعطل كود سابق.
3) إصلاح زر المزامنة في صفحات المطعم وربطه بطبقة مزامنة موحدة.
4) إضافة ملف restaurant-emergency-fix.js كطبقة حماية عامة للدرج الجانبي، المزامنة، وإخفاء أي بقايا كود مطبوعة.
5) تحديث sw.js بإصدار كاش جديد حتى لا يبقى المتصفح يستخدم النسخة القديمة المعطوبة.

مهم بعد رفع النسخة:
- افتح الموقع مرة واحدة ثم اعمل تحديث قوي للصفحة.
- إذا كان الموقع مثبت كتطبيق PWA، احذف التطبيق وثبته من جديد أو امسح بيانات الموقع حتى يتغير كاش Service Worker.
