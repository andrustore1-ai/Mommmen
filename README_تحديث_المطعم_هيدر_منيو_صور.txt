تحديث المطعم النهائي:
- صفحات المطعم الإدارية تستخدم نفس هيدر وقائمة النظام الرئيسي.
- زر مزامنة المطعم الداخلي أزيل، والاعتماد على زر المزامنة الرئيسي في الهيدر.
- المنيو الرقمي العام صار بواجهة حديثة شبيهة بكود Echo Store مع سلة، بحث، أقسام، بانر، وشعار.
- إعدادات المنيو تشمل اسم المطعم، اللون، الشعار، البانر، ورابط المنيو.
- يمكن إدخال رابط صورة أو رفع صورة من الجهاز للمنيو والبانر والأطباق إلى Firebase Storage.
- تم تثبيت لون النص الأبيض داخل البطاقات الملونة.
- صلاحيات الموظفين تحتوي أقسام المطعم والمطبخ والمنيو.
